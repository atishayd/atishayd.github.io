Project Description

This project implements an Optical Character Recognition (OCR) system to detect numbers from images. By utilizing advanced C++ techniques, it performs calculations such as horizontal and vertical symmetry, Euler number, and other image-processing metrics to isolate and recognize numbers. This system is especially useful in recognizing numerical patterns in bitmap images by calculating various geometric and mathematical properties, allowing robust detection even with noise or varying image conditions.

Use Cases

  • Automated Document Processing: Detecting numbers on scanned financial documents or forms.
  • License Plate Recognition: Identifying and segmenting numbers from license plate images.
  • Survey Data Entry: Recognizing handwritten or printed numbers for statistical data entry tasks.
  • Industrial Sorting: Distinguishing different products based on numerical labels from image captures.

Symmetry Calculations

int computeHorizontalSymmetry() {
    int symmetry_count = 0;
    for (int r = 0; r < height / 2; r++) {
        for (int c = 0; c < width; c++) {
            if (isSymmetricPixel(r, c)) symmetry_count++;
        }
    }
    return symmetry_count;
}

Counts horizontally symmetric pixels to calculate the symmetry score, aiding in differentiating circular or symmetrical digits like '0' and '8'.

Euler Number Calculation

int calculateEulerNumber() {
    int components = 0;
    int holes = 0;
    for (auto& blob : blobs) {
        if (blob.isHole()) holes++;
        else components++;
    }
    return components - holes;
}

Calculates Euler number by counting components and holes, helping to identify numbers with loops like '6' and '8'.

Vertical Symmetry Analysis

int computeVerticalSymmetry() {
    int symmetry_score = 0;
    for (int r = 0; r < height; r++) {
        for (int c = 0; c < width / 2; c++) {
            if (isSymmetricVertically(r, c)) symmetry_score++;
        }
    }
    return symmetry_score;
}

Evaluates vertical symmetry score to distinguish asymmetrical digits, aiding in differentiating shapes like '3' and '5'.
